Security Through
Invisibility

A secure medical data storage and transfer system combining AES-256 encryption with advanced LSB steganography to make patient records invisible to unauthorized parties.

🔒 Encode Data

Embed sensitive patient records securely into a medical image (e.g. DICOM PNG) using AES-256-GCM and randomized LSB steganography.

🔓 Decode Data

Extract and decrypt patient data from a stego-image using the patient's MRD number. Secure and instantaneous.
